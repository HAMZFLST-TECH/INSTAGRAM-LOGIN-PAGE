# Instagram Login Page UI 📸

A pixel-perfect recreation of the Instagram login page built using **pure HTML5 and CSS3**. This project focuses on layout precision, responsive design, and replicating a modern user interface without external libraries or frameworks.

## ✨ Features
* **Pure HTML/CSS:** No frameworks, no JavaScript—just clean, semantic code.
* **Fully Responsive:** Optimized for desktop, tablet, and mobile screens.
* **Pixel Perfect:** Careful attention to typography, spacing, and brand colors.
* **GitHub Pages:** Deployed directly from the repository.

## 🛠️ Built With
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, Flexbox, and Media Queries for responsiveness.
